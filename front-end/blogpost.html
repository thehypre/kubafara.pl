About

This website was created by me to apply in practice skills learned while getting AWS Cloud Practicioner certification. The goal was to create a CV webpage with a simple functionality of a view counter implemented in a not so simple AWS enviroment.

The first step was setting up a static page using S3 bucket and a Cloudfront distribution. The page redirects all HTTP traffic to HTTPS.  Using external domain provider i purchased a .pl domain and set it up with cloudfront using CNAME records added at home.pl admin panel. The website is based on a modyfied custom template from templatefilp.com

In the second step I manually implemented a simple JavaScript function on the frontend side and a Python Lambda Function on the backend to get a value from a dynamoDB database via Function URL and increment it by one.

The final step was to set-up most of the things again this time using IaaC with a CI/CD pipeline. I choose the terraforme since it is not only an Amazon service, but is as well used by other cloud providers. The CI/CD pipeline made in GitHub Actions provides a sufficient level of automation - for the fronted it updates the webpage contents in the S3 bucket and invalidates the cloudfront so any changes are too be seen by an user in a matter of seconds. The backend part handles and applies the terraform changes to the infractucture while ensuring the quality of the code.

Creation of the website was inspired by Cloud Resume Challenge. Working on this simple project show me how much is it yet to learn in the cloud. Even a simple view counter can be challenging for a person new to the AWS Services, however this only made me more eager to learn more, even in this projects i could still try to do things differently: try to build cloudfront in the terraform even with an external provider, create a new bucket to store terraform state more securly, try to use API Gateway instead of Function URL, include tests for the backend and frontend etc. Although for the next challenge i would like to build a proper web app and include those and different topics like advanced networking and security in the cloud.
